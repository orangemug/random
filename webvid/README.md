# README

Add the script to your path, and the usage is:

    ./webvid [input_video] [image] [title] [video_dir] [web_path] [dimensions]

Example:

    ./webvid.rb ~/video/input.ogv ~/video/block.png "test" "videos" "videos" "200x200"
